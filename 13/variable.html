변수(variable) - 프로그램을 실행하는 동안 값이 여러 번 달라질 수 있는 데이터 상수(constant) - 한번 지정하면 바뀌지 않는 데이터 나이 = 올해 연도 - 태어난 연도 + 1 나이, 올해 연도, 태어난 연도 - 변수 1 - 상수 변수선언 규칙 1. 변수 이름은 영어 문자와 언더스코어(_), 숫자를 사용 - 변수 이름의 첫 글자는 영어 대소문자나 언더스코어(_)만 쓸 수 있고, 숫자나 기호, 띄어쓰기는 허용하지 않음 now, _now, now25, now_25 - 사용할 수 있음 25now, now 25, *now - 사용할 수 없음 2. 자바스크립트는 영어 대소문자를 구별하며 예약어는 변수 이름으로 쓸 수 없음 - number와 Number, NumBer 모두 다른 변수 이름으로 인식하므로 구별해서 사용해야 함 - var같이 미리 정해 놓은 예약어는 변수 이름으로 사용할 수 없음 3. 여러 단어를 연결한 변수 이름은 중간에 대문자를 섞어씀 - 주로 한 단어로 이루어진 변수 이름은 모두 소문자로 쓰고, 두 단어 이상인 변수 이름은 totalArea나 TotalArea, Total_Area처럼 중간 에 대문자를 섞어 사용(낙타 표기법(camel case)) 4. 변수 이름은 의미 있게 작성 - 변수의 이름만 보고도 대충 어떤 값인지 추측할 수 있도록 하는 것이 좋음. ex) 시험 점수 합계 - total 평균 점수 - average a1, a2, a3, ....(x) var 변수명 var currentYear; // 올해 연도 var birthYear; // 태어난 연도 var age; // 나이 var currentYear, birthYear, age; --------------------------------------------------------------------- var birthYear; // 태어난 연도 변수 선언 birthYear = 2000; // 변수에 값 할당 var birthYear = 2000; // 태어난 연도 변수 선언과 동시에 값 할당하기
