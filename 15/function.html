여러 동작을 묶은 덩어리, 함수 - 함수(function): 프로그램의 기능별로 여러 명령을 묶어 놓은 것 - 내장 함수 : 자바스크립트에 미리 만들어 놓은 함수 (alert(), confirm(), prompt() 등) 함수의 선언 및 호출 - 함수가 어떤 명령을 처리할지 미리 알려 주는 것을 '함수를 선언한다' 또는 '함수를 정의한다'고 함 - 기본형 function 함수명() { 명령 } - 함수는 선언 및 정의만으로 실행이 되지 않는데 선언한 함수를 사용하는 것을 '함수를 호출한다' 또는 '함수를 실행한다'라고 함 - 기본형 함수명() 또는 함수명(변수)
